# TwinMind AI — Product Steering

Product: AI-powered Digital Twin for predictive maintenance in manufacturing

Users: manufacturing plant engineers, operations managers

Key workflows:
1. Monitor factory — real-time sensor data from machines streamed via WebSocket
2. Detect degradation — ML models flag anomalies and rising failure probability
3. AI diagnoses — Gemini-powered agent explains root cause and recommends action
4. Blockchain logs it — immutable on-chain record of every WARNING/CRITICAL event
